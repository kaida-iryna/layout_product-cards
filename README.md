# Product Cards Layout

E-commerce product card grid — responsive layout built with CSS Grid and Flexbox, part of Mate Academy HTML + CSS Advanced module.

## 🚀 Live Demo

🔗 https://kaida-iryna.github.io/layout_product-cards/

## 🛠 Tech Stack

- HTML5 (semantic structure)
- CSS3 (Grid + Flexbox)
- SCSS (Sass)
- BEM methodology

## 📚 What I learned

- CSS Grid for complex card layouts
- Combining Grid with Flexbox for nested components
- Responsive product card behavior across breakpoints
- Reusable BEM components for scalable e-commerce UI

## 🏃 Run locally

```bash
git clone https://github.com/kaida-iryna/layout_product-cards.git
cd layout_product-cards
# Open index.html in browser
```
